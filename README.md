# quest
a javascript assign "game"

The default will test your knowledge of the solar system,
drag tiles for the moons to the right planet.

This will only work with a minimal web-enviroment...

Alternative data can be used by invoking with e.g. ?name=anime
for example to assign anime characters to their correct series.
